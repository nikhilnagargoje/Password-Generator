Password Generator
A lightweight, responsive web application designed to generate strong, secure random passwords instantly. This tool helps users enhance their digital security by creating complex strings that are difficult to crack.

🚀 Live Demo
https://nikhilnagargoje.github.io/Password-Generator/

✨ Features
Instant Generation: Create a high-entropy password with a single click.

Copy to Clipboard: Quickly copy the generated password using the built-in icon.

Responsive Design: Optimized for mobile, tablet, and desktop views.

Modern UI: A clean, dark-themed interface with vibrant pink and green gradients.

🛠️ Tech Stack
HTML5: Structured the generator interface.

CSS3: Custom styling with gradients and responsive layouts.

JavaScript (ES6): Core logic for randomizing characters and handling DOM updates.

📂 Project Structure
Plaintext
├── index.html      # Main entry point
├── style.css       # Custom styling and layout
└── script.js       # Password generation logic

📖 How to Use
Visit the Site: Go to the Live Demo.

Generate: Click the "Generate Password" button to create a new random string.

View: The password will instantly appear in the display box.

Copy: Click the copy icon inside the password field to save it to your clipboard.
